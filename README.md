## How to start the project 

### Frontend
1) cd client
2) npm install
3) npm run dev

### Backend
1) cd server
2) npm install 
3) npm start


## Tech Stack

1. ReactJs
2. ThreeJs
3. NodeJs Express
4. React Color Picker
5. OpenAI for Image generation
6. React File Picker
7. Frame Motion
8. Deployment using Netlify and Render
